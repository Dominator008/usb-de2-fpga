USB DE2-FPGA VHDL

Hardware interface for USB controller on DE2 FPGA Platform

Download this project as a .zip file Download this project as a tar.gz file

Using only VHDL to develop a standalone (NIOS II free) interface to the ISP1362 USB device present on Altera DE2 Education board.

The interface is designed to replace legacy uart connections that are used for interfacing with PC, and utilize a more reliable, faster, and versatile USB 2.0 connection.

Design achievements:

  • Maximum speed: FMAX of 420MHz - limit due to minimum period restriction (max I/O toggle rate)
  • Low area: 339 LUTs (much lower than Nios-II implementation)
  • Extendibility: design is split in multiple files, to support adding more features if necessary.
  • Simple I/O interface: 8 bit data and one bit Ready signals incoming /outgoing data.

Design future work

  • Transfer rate: USB interface supports speed up to 1MBps - current design only transfers 1 byte per USB transaction, which translates to large protocol overhead and lower transfer rates.
