Usb-de2-fpga

Hardware interface for USB controller on DE2 FPGA Platform

Download this project as a .zip file Download this project as a tar.gz file

Using only VHDL to develop a standalone (NIOS II free) interface to the ISP1362 USB device present on Altera DE2 Education board. The interface is designed to replace legacy uart connections that are used for interfacing with PC, and replace with highly reliable, faster, and more versatile USB 2.0 connection.
